Â¿QUE SON Y COMO FUNCIONAN LAS PROPS EN REACT?


las props son valores que recibe un componente hijo de un componente padre. se agrupan en un objeto llamado PROPS, el cual puede recibir diferentes tipos de datos, como :

Strings
Numbers
Booleans
Arrays
Objects
Functions
React Elements
React Components

OJOO!!! LAS PROPS SON INMUTABLES, son valores de solo lectura

las props funcionan comoa atributos para jsx

- creacion de propiedades por defecto en los componentes
ej: 


const Mycomponent = props => {
    return(
        <div>{props.primeraPorDefecto}</div>
    )
}

Mycomponent.defaultProps = {
    primeraPorDefecto : "soy una prop por defecto",
}
